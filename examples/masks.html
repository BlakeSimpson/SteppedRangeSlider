Stepped Range Slider: Units and Masks

Please view source code for details. Here you will learn a simple method of adding units (such as $ or %) to your true slider values.

For further control, adding a display mask will present a custom value to the user while preserving the real value for your hidden input fields.

Display units and masks cannot be used togeather. Masks will take precedence over a unit.

Simple Units: Before: $

Add the option: display_unit: "$"

Simple Units: After: %

Add the options: display_unit: "%" and display_unit_place: "after"

Display Masks

Add the option: display_masks: mask_array

An array of "masks" may be provided which will be displayed on the page, while any input fields still recieve the true values

In addtion, a custom divider can be set, using: divider: " up to "
